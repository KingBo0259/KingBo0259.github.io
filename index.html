常见问题

我们整理了用户最常遇到的问题和解决方案，希望能帮助你快速解决问题

你可以通过以下方式更新App：

  • App Store（iOS）：打开App Store → 点击右上角头像 → 找到我们的App → 点击更新
  • Google Play（Android）：打开Google Play商店 → 点击菜单按钮 → 我的应用和游戏 → 找到我们的App → 点击更新
  • 自动更新：你也可以在应用商店设置中开启自动更新，确保你的App始终保持最新状态

如果你无法登录账户，请尝试以下步骤：

  1. 检查网络连接是否正常，尝试重新连接Wi-Fi或切换到移动数据
  2. 确认你输入的用户名/邮箱和密码正确，注意大小写和特殊字符
  3. 尝试通过"忘记密码"功能重置密码
  4. 清除App缓存数据（设置 → 应用管理 → 找到我们的App → 清除缓存）
  5. 如果以上方法都无效，请联系我们的客服团队

我们的App提供自动和手动两种数据备份方式：

自动备份：

在设置中开启"自动备份"功能，App会在你连接Wi-Fi且电量充足时自动备份数据到我们的服务器

手动备份：

点击设置 → 备份与恢复 → 立即备份，即可手动创建数据备份

恢复数据：

在新设备上登录你的账户后，App会自动提示你恢复数据，或者你也可以在设置 → 备份与恢复 → 选择备份版本进行恢复

如果App运行缓慢，你可以尝试以下优化方法：

  • 关闭其他正在运行的应用程序，释放系统资源
  • 清除App缓存数据（设置 → 应用管理 → 找到我们的App → 清除缓存）
  • 确保你的设备有足够的存储空间，建议至少保留2GB以上的可用空间
  • 更新App到最新版本，新版本通常包含性能优化
  • 重启你的设备，有时候简单的重启可以解决很多性能问题
  • 如果问题仍然存在，请检查你的设备是否满足App的最低系统要求

取消订阅或申请退款的步骤如下：

取消订阅：

iOS：设置 → 你的账户 → 订阅 → 找到我们的App → 取消订阅

Android：Google Play商店 → 菜单 → 订阅 → 找到我们的App → 取消订阅

请注意，取消订阅后，你仍然可以使用服务直到当前计费周期结束

申请退款：

如果你对购买的服务不满意，可以在购买后48小时内通过以下方式申请退款：

iOS：App Store → 账户 → 购买记录 → 找到我们的App → 报告问题 → 选择退款原因

Android：Google Play商店 → 菜单 → 订单历史 → 找到我们的App → 请求退款

超过48小时的退款请求，请联系我们的客服团队说明情况
